Setup guide:

Extract the zip folder to get the project folder.

Project:
    The project needs python3, flask 1.0.2 and a web browser to run.
    It needs internet connection to load jquery, d3 and d3-tooltip.
    Run app.py in python3 console.
    In a broswer open http://127.0.0.1:5000/
Experiments:
    experiments.py need numpy 1.15.4 to run.
    To run experiments run experiments.py in python3 console.
